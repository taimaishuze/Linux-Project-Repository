FrontRowSquad's Page

A Prosody chat server is the service our team decided to go with for our final project
We have a GitHub project you can check out to learn about how we deployed our service.
You can test it out with a chat client, such as Pidgin, and simply create a new user using the server info.

Try It Out!

You can try out the chat service right now by installing an XMPP client such as Pidgin On Ubuntu based systems you can do it by running "sudo apt-get install pidgin".
Or try a web based client such as Inverse
We have self registration enabled, so you can create a new account right from the client!
